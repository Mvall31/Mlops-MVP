# LAB 2 - Product Testing

Laboratorio de experimentacion con datos de producto, basado en notebooks y ficheros CSV de entrenamiento, prueba y predicciones.

## Contenido

- `Product_testing.ipynb`: notebook principal del laboratorio.
- `train (1).csv`: conjunto de entrenamiento.
- `test.csv`: conjunto de prueba.
- `predictions.csv`: salida generada con las predicciones.
