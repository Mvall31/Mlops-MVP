# LAB 4 - MLflow Tracking and Model Registry

Laboratorio de MLOps centrado en MLflow para tracking de experimentos, registro de modelos, versionado y despliegue de APIs de inferencia.

## Contenido

- `MLflow-Iris-Pipeline`: pipeline completo con el dataset Iris, tracking, registro, alias y API de prediccion.
- `MLflow-Diabetes`: entrenamiento, registro y evaluacion de una API para el dataset de diabetes.
